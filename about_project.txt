Данный тестовый проект создан с помощью языка программирования Python и фреймворка Django. 
При создании использовалась база данных sqlite3. 
Взаимодействие с проектом происходит через веб-интерфейс, поэтому все интуитивно понятно.
Для запуска проекта необходимо ввести команду: python manage.py runserver.
После запуска проекта, с помощью, вышеуказанной команды, тестовый сервер можно открыть по ссылке: http://127.0.0.1:8000/
